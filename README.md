## Project Description
This project analyzes Portuguese bank marketing campaign data to predict term deposit subscriptions using a Decision Tree classifier.

## Dataset
- **Source:** Kaggle - Bank Marketing Dataset
- **Records:** 11,162 customers
- **Features:** 17 attributes (demographics, financial, campaign variables)
- **Target Variable:** Term deposit subscription (yes/no)
## Methodology
- **Algorithm:** Decision Tree Classifier
- **Tools:** Python, scikit-learn, pandas, matplotlib, seaborn
- **Evaluation Metrics:** Accuracy, Precision, Recall, F1-Score, ROC-AUC

## Key Results
- Overall Accuracy: 89%
- Most Important Features: Duration, Balance, Age
